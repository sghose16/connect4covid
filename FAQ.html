FAQ

Q. If a game has reached max capacity, can I still join?

A. No, you will have to create a new game.

Q. Can I add a new group?

A. Yes, you may add a new group.

Q. If a game is currently in session, can I still join?

A. If it is available in the join tab, you can join. Otherwise, the host will have to let you join.

Q. How many games are available?

A. There are 7 games available.

Q. Can I add a new game?

A. No, you cannot add a new game. This is a feature we might add on later.

Q. Will the discord links be different for each game in progress?

A. Yes, each game will have its own discord link.
